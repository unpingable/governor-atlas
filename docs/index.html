claimdocs verifies declared bases, not truth itself. A basis may resolve or match a structural check; whether it adequately supports an edge is a separate human admission. Rendering preserves claim mode and never upgrades it.

The default view shows only the witnessing modes. Toggling the rest inflates the graph — that delta is the finding: what is real, what is planned, what is merely derivable.

The claim graph (cases/ + receipts/) is the artifact. This site is a readout. Delete docs/data and re-render — nothing of value is lost.
